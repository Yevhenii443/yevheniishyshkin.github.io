02. Experience


Hello! My name is Yevhenii and I enjoy creating programs. My interest in programming started back in 2019 when I decided to try myself as a student in a Unit Factory.


Unit Factory or School 42 - the place where my passion about programming increased every day. This is not an ordinary school where you have head-teacher or mentors, which can help you with your tasks.


Unit provides their own studying method called P2P(Peer to Peer). This method is unique, it teaches you not only soft skills and doing you communicable, also, it teaches you how to explain difficult things about programming to other, so you can more deeply understand how the whole process works by teaching people.


My whole path in Unit took me for almost 3 years.

The first stage of studying was Pool.

Pool - is a 1-month of difficult tasks on C language for every day. The last project of this month is to create a game on C using SDL2 or ncurses engines. I and two other students decided to create a team and distribute tasks. My task was to create design and make the main window with animation. At the end of a pool, we introduce our game to students, and they give us a feedback with marks. If you pass all the requirements and gather good amount of marks, you go on to CTrack.


CTrack - is a 6-month program, where your main tasks are to write algorithms and recreate internal functions of C language. After passing CTrack you have to pick one from ["FrontEnd", "Embedded", "DevOps", "GameDev"] branches. My choice was DevOps.


This branch was divided by 3 parts.

1. Python Start - learning Python basics.

2. Python Pro - learning Python OOP and different useful modules.

3. Docker Part - learning how to use Docker.

At the end of each part there was an "endgame" project, you can find them on my GitHub.
